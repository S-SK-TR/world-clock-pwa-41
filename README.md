# Dünya Saatleri PWA

Güncel Türkiye saatine göre dünya satlerini analog olarak gösteren Progressive Web App.

## Özellikler

- Türkiye saatini referans alarak tüm dünya saatlerini gösterme
- Analog saat arayüzü ile modern kullanıcı deneyimi
- Zaman dilimi ekleme/çıkarma işlevselliği
- Gün ışığı ve gece modu desteği
- Offline kullanım desteği
- Responsive tasarım ve mobil uyumluluk
- PWA özellikleri (manifest.json ve service worker)

## Kurulum

1. Depoyu klonlayın
2. `npm install` komutunu çalıştırın
3. `npm run dev` ile geliştirme sunucusunu başlatın

## Kullanım

- Ana ekranda Türkiye saatini ve varsayılan saatleri göreceksiniz
- Saat eklemek için 'Zaman Dilimi Ekle' bölümünü kullanın
- Saatleri kaldırmak için '×' butonuna tıklayın
- Gece modunu etkinleştirmek için sağ üst köşedeki butona tıklayın

## Testler

`npm test` komutu ile testleri çalıştırabilirsiniz.
